## 🧠 Key Concepts This Week (Quick Summary)

### What is Machine Learning?
Machine Learning is teaching a computer to learn from data — instead of writing every rule yourself.

> **Simple analogy:** Instead of teaching a child every single rule of grammar, you just show them thousands of sentences. They figure out the rules themselves. ML works the same way with data.

### The 3 Types of ML

| Type | Simple Explanation | Example |
|------|-------------------|---------|
| **Supervised** | Learn from labeled data (you know the right answers) | Spam detection |
| **Unsupervised** | Find patterns without labels | Customer grouping |
| **Reinforcement** | Learn by trial and reward | Game-playing AI |

### The ML Workflow
```
1. Collect Data → 2. Clean Data → 3. Train Model → 4. Evaluate → 5. Predict
```
